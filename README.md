# Smart Utility Toolkit 🛠️

A comprehensive Flutter mobile application featuring multiple utility tools in one convenient package.

## 🌟 Features

- **📱 Calculator** - Advanced calculator with history
- **⚖️ BMI Calculator** - Track your Body Mass Index with visual charts
- **💰 Bill Splitter** - Split bills easily among friends
- **📝 Notes** - Create, edit, and search through your notes
- **✅ Tasks** - Manage your to-do list with priorities and filters
- **🔄 Unit Converter** - Convert between different units
- **⚙️ Settings** - Customize themes and preferences

## 🏗️ Architecture

Built with **Clean Architecture** principles:
- **BLoC** for state management
- **Hive** for offline-first local storage
- **Injectable** for dependency injection
- **Freezed** for immutable data classes

## 🚀 Getting Started

### Prerequisites
- Flutter SDK >=3.19.0
- Dart SDK >=3.3.0
- Android Studio / VS Code

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/smart-utility-toolkit-flutter.git
cd smart-utility-toolkit-flutter
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```

## 🧪 Testing

Run all tests:
```bash
flutter test
```

## 📦 Build

### Android APK
```bash
flutter build apk --release
```

### Android App Bundle
```bash
flutter build appbundle --release
```
